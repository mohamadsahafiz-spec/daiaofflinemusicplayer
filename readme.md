# 🎵 Daia

**Daia** is a beautiful offline music player PWA with lyrics, sleep timer, crossfade, and custom EQ. Install it to your phone and play your music without internet.

### ✨ Features
- **100% Offline** - Add a folder of music and play anywhere
- **Neon Glass UI** - Dark/light theme with glassmorphism and animations
- **Lyrics Support** - Paste .lrc lyrics or type them manually, synced to song
- **Sleep Timer** - Auto pause after 5/15/30/60 minutes
- **Crossfade** - Smooth 3s transitions between songs
- **EQ Presets** - Flat, Bass Boost, Pop, Rock, Jazz, Club, Classical
- **Playlists** - Create and manage your own playlists
- **PWA Install** - Add to Home Screen on iOS & Android with custom icon
- **Heartbeat Loading** - Pulsating logo + splash screen on app open

### 📱 Install Daia

1. Open the GitHub Pages link on your phone
2. Tap `Share` > `Add to Home Screen`
3. Launch Daia from your home screen like a native app

### 🚀 How to Use

1. Tap **+ Add Music Folder** and select a folder with mp3 files
2. Tap any song to play
3. Use the player bar for play/pause, seek, shuffle, repeat
4. Paste lyrics in the Lyrics card and hit **Save** to store them
5. Set a sleep timer from the dropdown

### 🛠️ Tech Stack
- Vanilla HTML, CSS, JavaScript
- Web Audio API for EQ and Crossfade
- LocalStorage for playlists and lyrics
- Service Worker for offline PWA support
